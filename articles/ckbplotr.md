# Get started with ckbplotr

## Introduction

The `ckbplotr` R package provides functions to help create and style
plots in R using the [ggplot2](https://ggplot2.tidyverse.org/) graphics
package.

## CKB style

Use `+ ckb_style()` to apply CKB style to a ggplot. See [CKB style
ggplot](https://neilstats.github.io/ckbplotr/articles/ckb_style.md) for
details.

## Plots

Create plots of estimates against risk factor levels using
[`shape_plot()`](https://neilstats.github.io/ckbplotr/reference/shape_plot.md).
See [Shape
plots](https://neilstats.github.io/ckbplotr/articles/shape_plots.md) for
details.

Create forest plots using
[`forest_plot()`](https://neilstats.github.io/ckbplotr/reference/forest_plot.md).
See [Forest
plots](https://neilstats.github.io/ckbplotr/articles/forest_plots.md)
for details.

## Saving

See [Save plots to
files](https://neilstats.github.io/ckbplotr/articles/save_plots.md).

## More
